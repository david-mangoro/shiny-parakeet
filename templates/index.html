💡 How ScholarPulse Works

                
            

            

                

                    

                        
🔍

                        

                            
1. Live CORE API Feeds

                            
When you click Refresh Feed, the backend queries the official open-access CORE API v3 database to retrieve the latest and most relevant peer-reviewed works in Technology-Enhanced Learning and Open Education.

                        

                    

                    

                        
⚙️

                        

                            
2. Tone & Length Customization

                            
Clicking ⚡ Convert opens script generators. You can customize the tone (e.g. Casual, Academic, Edutainment) and length (60s Short, 3-5m Video, 10m deep dive) to dynamically update script outputs.

                        

                    

                    

                        
🔊

                        

                            
3. Text-to-Speech Preview

                            
Click 🔊 Listen to have your browser narrate the script automatically. It filters out host markers and director annotations to read only clean, conversational content.

                        

                    

                    

                        
📖

                        

                            
4. Citations & Social Sharing

                            
Click 🐦 Tweet to immediately draft a tweet with key findings, or click 📖 Cite to grab pre-formatted APA, MLA, and BibTeX citations for your reference manager.
